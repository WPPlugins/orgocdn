A WordPress plugin that optimizes all images on your WordPress site. Reduce images size up to 99% without losing any quality. For free

== Description ==
Speeding up your WordPress website has never been this simple.
We optimize your images without losing any of the quality.

= How we increase load-time without losing any quality =
1. We optimize all your images in your media gallery
2. Automatically replaces the images on your site with the new ones.
3. Reduce loading time up to 99%.

= Secure and reliable =
* All your original images will still be kept safe - we don't touch them.
* We create copies of your original images and then optimize them.

= We are here for you - Support =
We have an excited support team ready to help you and answer your questions and requests. [Contact us here](//wordpress.org/support/plugin/orgocdn).

== Installation ==
1. Upload ORGOTECH to the `/wp-content/plugins/` directory or install it via the Plugins menu in WordPress
2. Activate the plugin through the Plugins menu in WordPress
3. Open the WordPress admin menu and find ORGOTECH on the left
4. Click the big green button, and you're done!

== Frequently Asked Questions ==
= How many image can i optimize per month? =
Every domain is limited to 100 images.

= Is ORGOTECH's image optimization free to use? =
Yes! Up to 100 images will always be free.

= How much faster will my website be? =
A lot! The images of a website is the main reason why a website takes time to load in. And if you reduce the images by 99%, the websites load time also is reduced.

= How do I view my stats? =
Once you've installed the plugin you'll find the ORGOTECH link in the menu bar to the left, that's where the panel with all your stats will be.
